## 3. Architectural Invariants Enforced

Every document produced by this factory strictly enforces the following WebWaka invariants:

1. **Nigeria First:** All monetary values are stored as `kobo` integers. Integrations prioritize local providers (Paystack, NIBSS, Termii, NIMC).
2. **Offline First:** All frontends are PWAs utilizing Dexie.js (IndexedDB) for local mutation queuing, syncing to Cloudflare D1 when connectivity is restored.
3. **Build Once, Use Infinitely:** No duplicated logic. Shared primitives (auth, billing, notifications, AI) must reside in `@webwaka/core`.
4. **Multi-Tenant Isolation:** Strict JWT-driven tenant scoping at the database query level.
5. **Edge Native:** All backends deploy to Cloudflare Workers (Hono) with D1 and KV.
