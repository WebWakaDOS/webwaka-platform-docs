## المبادئ السبعة الأساسية

يجب أن يلتزم كل سطر كود في WebWaka OS v4 بهذه المبادئ السبعة الأساسية:

1. **ابنِ مرة واحدة، استخدم بلا حدود** — جميع الميزات معيارية وقابلة لإعادة الاستخدام
2. **الجوال أولاً** — واجهة المستخدم محسّنة للجوال قبل سطح المكتب
3. **PWA أولاً** — دعم التثبيت والقدرات الأصيلة
4. **العمل دون اتصال أولاً** — يعمل بدون إنترنت عبر IndexedDB
5. **نيجيريا أولاً** — يدمج الخدمات المحلية (Paystack، NGN، Termii)
6. **أفريقيا أولاً** — دعم متعدد العملات والحدود
7. **ذكاء اصطناعي محايد للبائع** — يستخدم محرك التجريد CORE-5
